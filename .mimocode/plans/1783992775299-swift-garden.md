# EVOID v2 Rewrite Plan вЂ” Reference Runtime for IOP

> Based on RFC-0001. Intent is the platform. The language is an implementation detail.

## Overview

Complete rewrite of EVOID from a Python web framework to a **language-independent IOP runtime specification** with Python as the first reference implementation.

**Key changes:**
- Framework в†’ Runtime
- Monolithic в†’ Plugin-based
- Python-only в†’ Language-independent spec
- FastAPI-coupled в†’ Transport-agnostic
- Configuration в†’ Composition Root

## Core Components (Phase 1)

### 1. Intent System (`evoid/core/intent.py`)

```python
from enum import Enum
from dataclasses import dataclass, field
from typing import Any

class IntentLevel(str, Enum):
    EPHEMERAL = "ephemeral"
    STANDARD = "standard"
    CRITICAL = "critical"

@dataclass(frozen=True)
class Intent:
    """Intent is immutable. It carries purpose, not behavior."""
    name: str
    level: IntentLevel
    metadata: dict[str, Any] = field(default_factory=dict)

class IntentRegistry:
    """Global registry of intent definitions."""
    def register(self, intent: Intent) -> None: ...
    def resolve(self, name: str) -> Intent: ...
    def all(self) -> dict[str, Intent]: ...
```

### 2. Intent Resolver (`evoid/core/resolver.py`)

```python
from typing import Protocol, Any

class ResolverEngine(Protocol):
    def resolve(self, intent: Intent) -> dict[str, Any]: ...

class IntentResolver:
    """Maps Intent в†’ execution config. Language-agnostic."""
    def __init__(self, engine: ResolverEngine): ...
    def resolve(self, intent: Intent) -> PipelineConfig: ...
```

### 3. Pipeline (`evoid/core/pipeline.py`)

```python
from typing import Protocol, Any, Callable, Awaitable

Processor = Callable[["Context"], Awaitable[Any]]

@dataclass(frozen=True)
class PipelineConfig:
    processors: tuple[str, ...]  # Processor names
    priority: int
    timeout: float

class Pipeline:
    """Composes and executes processor chains."""
    def __init__(self, config: PipelineConfig, registry: dict[str, Processor]): ...
    async def execute(self, context: "Context") -> Any: ...
```

### 4. Execution Context (`evoid/core/context.py`)

```python
@dataclass
class Context:
    """Isolated execution environment for one Intent."""
    intent: Intent
    state: dict[str, Any]
    deps: dict[str, Any]          # Injected dependencies
    metadata: dict[str, Any]
    
    # Lifecycle hooks
    async def on_start(self) -> None: ...
    async def on_complete(self) -> None: ...
    async def on_error(self, error: Exception) -> None: ...
```

### 5. Runtime (`evoid/core/runtime.py`)

```python
@dataclass
class RuntimeConfig:
    name: str
    adapter: str           # "asgi", "cli", etc.
    engines: dict[str, str]  # {"schema": "pydantic", "storage": "sqlite", ...}
    processors: list[str]

class Runtime:
    """Main entry point. Assembles everything from config."""
    def __init__(self, config: RuntimeConfig): ...
    async def start(self) -> None: ...
    async def stop(self) -> None: ...
    async def execute(self, intent: Intent) -> Any: ...
```

## Plugin Contracts (Phase 2)

All engines implement Protocol classes:

```python
# evoid/contracts/schema.py
class SchemaEngine(Protocol):
    def validate(self, data: Any, schema: type) -> Any: ...
    def serialize(self, obj: Any) -> dict: ...
    def deserialize(self, data: dict, schema: type) -> Any: ...

# evoid/contracts/storage.py
class StorageEngine(Protocol):
    async def read(self, key: str) -> Any | None: ...
    async def write(self, key: str, value: Any) -> bool: ...
    async def delete(self, key: str) -> bool: ...
    async def health(self) -> bool: ...

# evoid/contracts/cache.py
class CacheEngine(Protocol):
    async def get(self, key: str) -> Any | None: ...
    async def set(self, key: str, value: Any, ttl: float) -> bool: ...
    async def delete(self, key: str) -> bool: ...

# evoid/contracts/adapter.py
class RuntimeAdapter(Protocol):
    async def start(self, runtime: "Runtime") -> None: ...
    async def stop(self) -> None: ...
    async def handle(self, raw_event: Any) -> Intent: ...
```

from starlette.applications import Starlette
from starlette.requests import Request
from starlette.responses import JSONResponse

class ASGIAdapter:
    def __init__(self, runtime: Runtime): ...
    
    async def handle(self, request: Request) -> JSONResponse:
        # 1. Convert request в†’ Intent
        intent = await self._extract_intent(request)
        # 2. Execute through runtime
        result = await self.runtime.execute(intent)
        # 3. Return response
        return JSONResponse(result)

## Key Principles

1. **Intent is permanent, infrastructure is temporary**
2. **Plugins communicate through contracts, never concrete implementations**
3. **Configuration is the Composition Root**
4. **Every language must behave identically**
5. **Start simple, upgrade only when it hurts (IOP Level 1 в†’ 4)**
