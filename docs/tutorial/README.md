# Tutorial - Step by Step

This tutorial shows you how to use **EVOID** step by step.

Each section builds on the previous ones, but is structured to separate topics so you can go directly to any specific one.

## How to Use

All code blocks can be copied and run directly.

To run any example, copy the code to a file and run:

```bash
evo run
```

Or for quick testing:

```bash
python main.py
```

## Prerequisites

- Python 3.13+
- EVOID installed: `uv add evoid`
